# Executive Dashboard Project

Executive Dashboard is a web application that provides data visualization. It allows users to filter, tune and if need bias the data being sent to the A.I modules for analysis.


## Technologies Used

- **Frontend:** HTML, CSS, JavaScript, ExpressJS
- **Version Control:** Git, GitHub

## Prerequisites

Before you begin, ensure you have the following installed:

- [Node.js](https://nodejs.org/en/download/): JavaScript runtime built on Chrome's V8 JavaScript engine.
- [Angular CLI](https://cli.angular.io/): Command line interface for Angular.
- [cors](https://www.npmjs.com/package/cors): CORS is a node.js package for providing a Connect/Express middleware that can be used to enable CORS with various options.

## Getting Started

To run the Executive Dashboard locally, follow these steps:

1. **Clone the Repository:**

`git clone https://github.com/rodAlm08/Executive_Dashboard.git`

2. **Navigate to the Project Directory:**

`cd exec-dash`

3. **Install Angular CLI (if not installed):**

`npm install -g @angular/cli`

4. **Install Application Dependencies:**

`npm install`

5. **Run the Application:**

`ng serve`

6. **Access the Application:**
Open your web browser and go to `http://localhost:4200` to access the Executive Dashboard.

## Usage

1. **Select Variables:**
From the dropdown choose the variables that should be sent to the AI modules for analysis.

2. **Send selected variables for analysis:**
After selecting all applicable variables, click the button to send the selected data to the AI module.
